# 💳 Анализатор кэшбэка (Cashback Analyzer)

Легковесное веб-приложение для тех, кто активно пользуется картами разных банков и устал каждый месяц сравнивать пересекающиеся категории кэшбэка в уме или Excel. 

Приложение позволяет удобно ввести предложенные банками категории, столкнуть их лбами в наглядном интерфейсе, выбрать самые выгодные и сгенерировать удобную памятку для телефона.

🌐 **Приложение работает полностью в браузере. Никакие данные не отправляются на сервер.**

## ✨ Главные возможности

* **Мультибанкинг:** Добавляйте любое количество банков, указывайте лимиты по кэшбэку и количество доступных для выбора мест (слотов).
* **Умное сравнение:** Объединяйте категории с разными названиями (например, «Фастфуд» от Банка А и «Рестораны и фастфуд» от Банка Б) в единые «Группы» для наглядного сравнения.
* **Автоподстановка (Smart Memory):** Приложение запоминает введенные вами лимиты, проценты и правила банков в `localStorage`. В следующем месяце вам достаточно начать вводить название, и все условия подтянутся автоматически.
* **Учет лимитов:** Автоматическая блокировка менее выгодных предложений банка, если вы уже выбрали в нем максимальное количество категорий.
* **Экспорт памятки в 3 форматах:**
  * 📋 **Текст:** Оптимизировано для отправки в Telegram или другие мессенджеры.
  * 🖨 **PDF:** Специальная мобильная верстка с крупным шрифтом (вызывается через стандартное окно печати).
  * 📊 **CSV:** Выгрузка в Excel для истории или сложной аналитики.
* **Drag-and-Drop импорт:** Перетяните CSV-файл за прошлый месяц прямо в окно браузера для мгновенной загрузки списка категорий.

## ⚠️ Важно: О хранении данных

Поскольку приложение работает **полностью локально** и не имеет бэкенда, все ваши профили банков, лимиты и добавленные категории сохраняются исключительно в памяти вашего текущего браузера (`localStorage`).

* **Очистка истории:** Если вы полностью очистите данные сайтов в браузере (кэш и cookie), ваш словарь автоподстановок и лимиты **будут безвозвратно удалены**.
* **Разные устройства:** Данные не синхронизируются между вашим телефоном и компьютером или между разными браузерами (например, Chrome и Safari).
* **Режим инкогнито:** При использовании приватного режима настройки не сохранятся после закрытия вкладки.

💡 **Как не потерять данные (Резервное копирование):** Регулярно используйте кнопку **«📊 Экспорт в Excel»**. Она скачает актуальный список ваших категорий. Если данные в браузере когда-нибудь сбросятся, вы сможете мгновенно восстановить их, просто перетянув этот CSV-файл обратно на страницу анализатора (Drag-and-Drop).

## 🚀 Как использовать (Установка)

Приложение состоит всего из одного файла и не требует установки, Node.js, баз данных или сторонних библиотек.

**Вариант 1: Использование онлайн (Рекомендуется)**
Просто откройте страницу проекта (работает прямо в браузере):
👉 [Анализатор Кэшбэка](https://vadimlevo.github.io/Cashback-Analyzer/)

**Вариант 2: Использование офлайн**
1. Скачайте файл `index.html` из этого репозитория.
2. Откройте его в любом современном браузере (Chrome, Safari, Firefox, Edge).

## 🛠 Технологии

* **HTML5 / CSS3** — адаптивная верстка и стилизация печати (`@media print`).
* **Vanilla JavaScript** — вся логика работы, парсинг CSV и управление состоянием без использования фреймворков (React/Vue/и т.д.).
* **localStorage API** — надежное хранение пользовательских словарей и профилей прямо на устройстве.
